What are they?
--------------------------
media.py is a Python module that contains the Movie class.  entertainment_center.py is a Python program that uses
the media and fresh_tomatoes modules to generate an html file containing information about the movies, including
images and a link to their trailers

How to use
--------------------------
Place entertainment_center.py and media.py in the same directory. Run entertainmen_center.py
